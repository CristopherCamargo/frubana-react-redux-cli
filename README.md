## Install

`yarn add --global @frubana/react-redux-cli`

## Use

`react-redux-cli --[option] [module-name]`

example:

```
react-redux-cli -p products
```

optional you can pass target directory like second argument

example:

```
react-redux-cli -p products containers
```

### Options

```
  --atom,
  --molecule,
  --organism,
  --page,
  --template,
  --directory,
  --store,
  -a: --atom,
  -m: --molecule,
  -o: --organism,
  -p: --page,
  -t: --template,
  -s: --store
```
